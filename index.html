The-validator

Extension for Google Chrome to validate the HTML pages using W3C Validation service.

The Validator: Introduction.

This validator will validate the HTML pages using W3C validation service. Using this validator, we aim to create a extension using which we can validate our pages in just one click. No need to hit the url of W3C validator again and again. This extension will be created primarily for Google Chrome. Later on, we will extend it to Mozilla Firefox also.

Need of Validator

There are two primary reasons for developing this validator:
I) There were no validation extension for Google Chrome as WAT available in IE. There are certain issues with WAT also such as many times it would refuse to open the validator, it works in IE only which is for very obvious reasons have been obsolete.
II) Developing a extension, would give us more in depth understanding of working of browser and also provide us a better working knowledge of Javascript.

Current Status

Currently, we are facing the problem(kind of roadblock) with W3C validator. It is validating the URL infinitely in a loop again and again, which is causing further issues. I am doing more research into the working of W3C validator to slove this issue. I have come across many solutions such as URL hex encoding, or using a flag to stop the loop.
